
# 🌟 Interactive Personal Data Collector

Welcome to the **Interactive Personal Data Collector** project!  
This Python program collects personal information from the user and displays details like **data type** and **memory address** of each input.

---

## 🚀 Features

✅ Collects user information interactively  
✅ Takes input for:
- 👤 Name  
- 🎂 Age  
- 📏 Height (in meters)  
- 🔢 Favourite Number  

✅ Displays:
- Variable **type**
- Variable **memory address**
- Approximate **birth year**

---

## 🛠️ Technologies Used

- **Python 3**
- User Input Handling
- Data Types (`int`, `float`, `str`)
- Memory Address using `id()`
