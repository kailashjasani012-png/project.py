# 🧮 Data Analyzer & Transformer

> A **Python console-based** program that lets you input, analyze, filter, sort, and explore statistics on a 1D array of numbers — all through an interactive menu!

---

## ✨ Features

| # | Feature | Description |
|---|---------|-------------|
| 1️⃣ | **Input Data** | Enter a custom 1D array of integers |
| 2️⃣ | **Display Data Summary** | View count, min, max, sum & average |
| 3️⃣ | **Calculate Factorial** | Compute factorial using recursion |
| 4️⃣ | **Filter by Threshold** | Split data into above / below / equal groups |
| 5️⃣ | **Sort Data** | Sort in ascending or descending order |
| 6️⃣ | **Dataset Statistics** | Full statistical overview of your data |
| 7️⃣ | **Exit** | Gracefully quit the program |
